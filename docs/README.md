# Reino del Umbral

Proyecto de RPG multijugador 2D web-first, inspirado en el espíritu de los MMORPG clásicos de vista cenital, con foco en mundo persistente, progresión, interacción social, combate, exploración y escalabilidad técnica.

## Objetivo del proyecto

Construir una primera versión jugable en navegador donde un jugador pueda:

- registrarse e iniciar sesión,
- crear o seleccionar personaje,
- entrar a una zona de juego compartida,
- ver a otros jugadores,
- moverse,
- combatir,
- chatear,
- conseguir loot,
- conservar progreso.
